# Project Overview

This project is a modernized Craigslist-style marketplace interface built with SvelteKit. It focuses on a clean UI, smooth interactions, and modern features that improve the traditional classifieds experience.

## Key Features

- Full UI overhaul with a clean, glassy, modern layout
- Dynamic listing grid with sorting and advanced filters
- Per-listing detail view with a blurred background overlay
- Category sidebar, map filter modal, and collapsible filter panel
- Randomized geolocation for listings (demo mode)
- Responsive design for both desktop and mobile
